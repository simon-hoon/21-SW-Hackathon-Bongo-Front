## how to start
*expo* 환경에서 동작합니다.

코드를 다운받은 후 아래와 같이 expo를 설치해주세요.
```
npm i expo
```

만약 중간에 오류가 날 경우 다음과 같이 업데이트해주세요.
```
expo update
```
